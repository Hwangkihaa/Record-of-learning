데이터(data)
- "주어진 어떤 것"을 의미하는 라틴어
- 데이터를 만들어 사용하는 사용자간에 합의된 형식으로 된 사실, 개념, 명령 등의 비트 표현
통신(Communication)
- 정보의 공유
데이터 통신(Data Communication)
- 특정 전송매체를 통한 두 장치간의 데이터 교환

데이터 통신의 기본 특성
- 전달, 정확성, 적시성, 파형 난조

데이터 통시신의 구성 요소
- 메시지, 송신자, 수신자, 전송매체, 프로토콜

데이터 표현
- 문자, 숫자, 화상, 음성, 동영상

데이터 전송 방식
- 단방향 방식, 반이중 방식, 전이중 방식

네트워크
- 전송매체에 연결된 노드와 링크의 집합

네트워크 평가 기준
- 성능, 신뢰도, 보안

네트워크 연결 유형
- 점대점 회선 구성, 다중점(멀티드롭)

네트워크 접속 형태
- 그물형(Mesh), 성형(Star), 버스형(Bus), 링형(Ring), 혼합형(Hybrid)

네트워크 분류 : 규모, 목적, 구조, 이용 기술에 따라 구분
- LAN(근거리 통신망)
- WAN(광역 통신망)
- MAN(도시 통신망)
- PAN(개인 영역 통신망)
- BAN(신체 통신망)

인터넷(INTERNET)
- 수많은 정보를 간편하게 이용할 수 있게 해주는 통신 시스템
- 정해진 구조를 가진 조직된 시스템

프로토콜
- 개체간에 이루어지는 데이터 통신을 규제하는 규칙들의 집합
- 주요 요소 : 구문(Syntax), 의미(Semantics), 타이밍(Timing)

OSI 모델
- Open System Interconnection
- 국제표준화기구(ISO)에서 제정한 개방시스템 상호연결 모델
- 개방 통신 / 안전한 상호 연동

TCP/IP
- 물리주소/논리주소/포트주소/특수주소
